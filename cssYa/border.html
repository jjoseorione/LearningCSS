Bordes y sus propiedades CSS

Descripción

Cuando trabajamos con HTML debemos hacernos a la idea de que todo elemento que se crea en un documento o página HTML genera una caja. Hasta ahora sabemos crear elementos (como div, span, h1, h2, p, etc.) y hemos notado que si agregamos la propiedad CSS background-color, todo un bloque rectangular (dentro del que está contenido el texto del elemento) se pinta de dicho color. El borde de ese bloque rectangular o caja es el borde del elemento.

Es importante tener claras las diferencias entre el border, el margin y el padding del elemento, los cuales se verán más adelante. Por ahora quedémonos con la idea de que el border es el borde de la caja que crea cada elemento en HTML

Las propiedades más importantes del borde a las que tenemos acceso son las siguientes:

  • border-width: Define el grosor del borde. Toma valores numéricos en la unidad deseada.
  • border-style: Define la forma que tendrá el borde. Entre los valores que puede tomar, se encuentran los siguientes:
    • none: No se muestra el borde.
    • solid: Se muestra una línea sólida y contínua.
    • dotted: Borde punteado (puntos redondos)
    • inset: Se muestra en 3D hacia adentro
    • outset: Se muestra en 3D hacia afuera
    • dashed: Línea discontínua.
    • double: Se muestran dos líneas sólidas y contínuas.
    • groove: Se muestra en 3D y doble, parecido al marco de un cuadro.
    • ridge: Igual a groove pero con colores invertidos.
    • dotted solid: Combinación de dos estilos. En este caso, se especifica que los bordes superior e inferior irán punteados y los costados serán sólidos.
    • dotted solid double dashed: Combinación de 4 estilos. El primer valor toma el lugar del borde superior y el resto siguen en sentido de las manecillas del reloj.
  • border-color: Especifica el color del que se muestra el borde.

Las propiedades de color y grosor, al igual que estilo, también pueden definirse de forma independiente para cada borde o en parejas para horizontales y verticales. Además, también es posibe definir el border de forma resumida con la propiedad border: tamaño estilo color. De esta forma definimos el tamaño, estilo y color en una sola línea de código CSS, pero sólo podemos definir un sólo color, un sólo estilo y un sólo grosor.

Ejemplo

Ejemplo de bordes:

<!DOCTYPE html>
<html>
<head>

<title></title>
<meta charset="UTF-8">
<style>
#titulo
{
border-width: 15px 5px;
border-style: dotted dashed;
border-color: green red blue purple;
}
.pregunta
{
background-color: yellow;
border: 2px solid black;
}
.respuesta
{
border: 1px dashed black
}
</style>

</head>
<body>

<h1 id="titulo">Cuestionario</h1>
<p class="pregunta">¿Quién descubrió América?</p>
<p class="respuesta">Cristobal Colón</p>
<p class="pregunta">¿En qué año fue?</p>
<p class="respuesta">1942</p>

</body>
</html>
