Definición de varias reglas para un mismo elemento HTML

Subtítulo

En ocasiones es necesario desglosar la inicialización de propiedades en distintas reglas. Esto debido a que podemos crear reglas generales que apliquen a varios elementos y luego ir creando reglas más particulares que sólo apliquen a algunos o a sólo un elemento.

Por ejemplo, supongamos que queremos que todos los elementos h de título usen fuente sans-serif, pero queremos que únicamente el elemento h1 sea de color azul. Se debería crear una regla general para todos los títulos y luego hacer una particular para el título h1.

Subtítulo

Tomando el ejemplo de los títulos descritos anteriormente, se tendría el siguiente código:

<!DOCTYPE html>
<html>
<head>

<title></title>
<meta charset="UTF-8">
<style>
h1, h2, h3, h4, h5, h6{
font-family: sans-serif;
}
h1{
color: blue;
}
</style>

</head>
<body>

<h1>Título de nivel 1</h1>
<h2>Título de nivel 2</h2>
<h3>Título de nivel 3</h3>
<h4>Título de nivel 4</h4>
<h5>Título de nivel 5</h5>
<h6>Título de nivel 6</h6>

</body>
</html>
