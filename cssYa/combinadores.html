Combinadores en CSS

Combinadores

Los combinadores en CSS especifican la forma en que se pueden relacionar dos o más selectores para crear así un selector más específico. Existen 4 combinadores diferentes en CSS.

  • Descendiente (espacio)
  • Hijo (>)
  • Hermano adyacente (+)
  • Hermano menor (~)

Combinador descendiente

Descripción

El combinador de descendiente sirve para indicar que todos los elementos específicos que sean hijos de otro elemento específico deben cumplir dicha regla. Para especificar la regla que deberá seguir el descendiente se deben de especificar el elemento ancestro y el elemento descendiente separados por un espacio.

Ejemplo

En este ejemplo se crea una regla para elementos p que sean descendientes de elementos div. El código es el siguiente.

<!DOCTYPE html>
<html>
<head>

<title></title>
<meta charset="UTF-8">
<style>
div p
{
background-color: yellow;
}
</style>

</head>
<body>

<p>Párrafo 0, no está dentro del div.</p>
<div>
<p>Párrafo 1, dentro del div.</p>
<p>Párrafo 2, dentro del div.</p>
<span><p>Párrafo 3, dentro de un span y dento del div.</p></span>
</div>
<p>Párrafo 4, no está dentro del div.</p>
<p>Párrafo 5, no está dentro del div.</p>
<p>Párrafo 6, no está dentro del div.</p>

</body>
</html>

Combinador hijo

Descripción

El combinador de hijo sirve para especificar la regla de estilo que recibirá un elemento específico que sea hijo de otro elemento específico. Un elemento que es hijo de otro es aquél que es descendiente directo. Para especificar la regla de estilo que tendrá el elemento hijo, se deben específicar el elemento padre y el elemento hijo separados por un símbolo mayor que (>).

Ejemplo

En este ejemplo se crea una regla para elementos p que sean hijos de elementos div. Se debe notar que el tercer párrafo es descendiene del div, pero no es su hijo y por eso no se le aplica la regla de estilo. El código es el siguiente.

<!DOCTYPE html>
<html>
<head>

<title></title>
<meta charset="UTF-8">
<style>
div > p
{
background-color: yellow;
}
</style>

</head>
<body>

<p>Párrafo 0, no está dentro del div.</p>
<div>
<p>Párrafo 1, dentro del div.</p>
<p>Párrafo 2, dentro del div.</p>
<span><p>Párrafo 3, dentro de un span y dentro del div.</p></span>
</div>
<p>Párrafo 4, no está dentro del div.</p>
<p>Párrafo 5, no está dentro del div.</p>
<p>Párrafo 6, no está dentro del div.</p>

</body>
</html>

Combinador hermano adyacente

Descripción

El combianador hermano adyacente sirve para especificar la regla que recibirá un elemento específico que sea hermano y sea adyacente a otro elemento específico. Es decir, que sea hijo del mismo padre y que se encuentre uno a continuación del otro. Para especificar la regla de estilo que tendrá el hermano adyacente se debe especificar el elemento hermano mayor y el elemento hermano menor separados por un símbolo de suma (+).

Ejemplo

En este ejemplo se crea una regla para elementos p que sean hermanos adyacentes de elementos div.

<!DOCTYPE html>
<html>
<head>

<title></title>
<meta charset="UTF-8">
<style>
div + p
{
background-color: yellow;
}
</style>

</head>
<body>

<p>Párrafo 0, no está dentro del div.</p>
<div>
<p>Párrafo 1, dentro del div.</p>
<p>Párrafo 2, dentro del div.</p>
<span><p>Párrafo 3, dentro de un span y dentro del div.</p></span>
</div>
<p>Párrafo 4, no está dentro del div.</p>
<p>Párrafo 5, no está dentro del div.</p>
<p>Párrafo 6, no está dentro del div.</p>

</body>
</html>

Combinador hermano menor

El combinador hermano menor sirve para especificar la regla que deberá deseguir un elemento específico que es hermano menor de otro elemento específico. Cuando se dice que un elemento es hermano menor de otro, significa que son ambos hijos de un mismo padre y que el menor aparece o está definido después que el mayor. Para especificar la regla de estilo que tendrá el hermano menor, se debe especificar el hermano mayor seguido del hermano menor y separados por una virguilla (~).

Ejemplo

En este ejemplo se crea una regla para elementos p que sean hermanos adyacentes de elementos div.

<!DOCTYPE html>
<html>
<head>

<title></title>
<meta charset="UTF-8">
<style>
div ~ p
{
background-color: yellow;
}
</style>

</head>
<body>

<p>Párrafo 0, no está dentro del div.</p>
<div>
<p>Párrafo 1, dentro del div.</p>
<p>Párrafo 2, dentro del div.</p>
<span><p>Párrafo 3, dentro de un span y dentro del div.</p></span>
</div>
<p>Párrafo 4, no está dentro del div.</p>
<p>Párrafo 5, no está dentro del div.</p>
<p>Párrafo 6, no está dentro del div.</p>

</body>
</html>
