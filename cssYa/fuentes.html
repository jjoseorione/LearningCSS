Propiedades relacionadas a fuentes

Descripción

Las propiedades relacionadas a las fuentes más importantes en CSS son las siguientes:

  • font-family Define la fuente que se aplicará al elemento HTML. Para que puedan visualizarse adecuadamente, las fuentes usadas deberán estar instaladas en el dispositivo en el que se muestra la página (o en su defecto ser cargadas en la página, de manera similar que una imagen). Las fuentes que se encuentran por default en todos los dispositivos son serif, sans-serif y monospace. Se puede agregar más de una fuente, en cuyo caso el navegador usará la primera que esté disponible en el dispositivo.
  • font-size Define el tamaño de la fuente. Entre otras formas (unidades) podemos definirlo en píxeles.
  • font-style Podemos indicar los valores normal, italic y oblique.
  • font-weight Entre otros, podemos indicar los valores normal, bold, bolder, lighter.
  • font-variant Font variant permite elegir en qué forma se ven las minúsculas en un texto. Con la opción "normal" las veremos como de costubre, mientras que con la opción "small-caps" las minúsculas se muestran como mayúsculas pequeñas.

Ejemplos

Se mostrarán distintos ejemplos de uso de fuentes. El código es el siguiente:

<html>
<head>

<title></title>
<meta charset="UTF-8">
<style>
h2 {
color: blue;
font-variant: small-caps;
}
h3{
font-family: sans-serif;
font-size:18px;
font-weight: bold;
}
h4{
font-family: serif;
font-size:12px;
font-style: oblique;
}
h5{
font-family: monospace;
font-size:16px;
font-style: oblique;
font-weight: bold;
}
</style>

</head>
<body>

<h2> Título de nivel 2 en fuente serif, color azul y variante small-caps </h2>
<h3>
Título de nivel 3 con fuente sans-serif, de 18 píxeles y peso bold (negrita).
</h3>
<h4>
Título de nivel 4 definido con fuente serif, de 18 píxeles y estilo oblique.
</h4>
<h5>
Título de nivel 5 definido con fuente monospace, de 16 píxeles, peso bold y estilo oblique.
</h5>
</div>
</body>
</html>
