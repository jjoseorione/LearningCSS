Agrupación de varios elementos HTML con una misma regla de estilo

Descripción

Esta característica nos permite ahorrar la estructura de reglas duplicadas para diferentes elementos de HTML. Para que una misma regla aplique a más de un elemento HTML es suficiente con agruparlos por comas.

Subtítulo

Ejemplo de código donde se usa la misma regla para :

<!DOCTYPE html>
<html>
<head>

<title></title>
<meta charset="UTF-8">
<style>
h1,h2,h3 {
font-family: sans-serif;
color: #0000ff;
}
</style>
</head>
<body>

<h1>Título de nivel 1</h1>
<h2>Título de nivel 2</h2>
<h3>Título de nivel 3</h3>

</body>
</html>
