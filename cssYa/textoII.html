Más propiedades relacionadas al texto

Descripción

Otras propiedades útiles relacionadas al texto son las siguientes:

  • letter-spacing: Permite indicar el espaciado que debe haber entre caracteres.
  • word-spacing: Permite indicar el espaciado que debe haber entre palabras.
  • text-indent: Esta propiedad define la sangría que debe agregarse a la primera línea de un bloque de texto. Se pueden agregar valores negativos para que le sangría sea negativa (a la izquierda).
  • text-transform:. Esta propiedad manipula la forma en la que se muestran mayúsculas y minúsculas en un bloque de texto. Se pueden tomar alguno de los siguientes valores:
    • capitalize: Inicia con mayúscula la primera letra de cada palabra.
    • lowercase: Convierte a minúsculas todas las letras del texto.
    • uppercase: Convierte a mayúsculas todas las letras del texto.
    • none: No provoca cambios.

Ejemplo

Ejemplo en el que se usan las propiedades descritas previamente:

<!DOCTYPE html>
<html>
<head>

<title></title>
<meta charset="UTF-8">

<style>
h1
{
letter-spacing: 10px;
word-spacing: 30px;
text-transform: capitalize;
}
h2
{
word-spacing: 10px;
text-transform: uppercase;
}
p
{
text-indent: 50px;
}
</style>

</head>
<body>

<h1>Esto es un título de nivel 1</h1>
<h2>Esto es un título de nivel 2</h2>
<p>
Esto es un párrafo. Puede notarse que únicamente la primera línea de texto tiene la sangría especificada, mientras que el resto de las líneas permanece a raya al inicio del renglón. De igual forma, puede notarse que el título de nivel 1 capitaliza todas las palabras y hace el espaciado correcto entre palabras y entre letras. Mientras tanto, el título de nivel 2 convierte todo el texto en mayúsculas y agrega el espaciado especificado entre palabras.
</p>

</body>
</html>
