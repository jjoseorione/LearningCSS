Propiedades relacionadas al texto

Descripción

Las propiedades más importantes relacionadas al texto son las siguientes:

  • color: Sirve para definir el color del que se mostrrá el texto del elemento. El color puede ser definido en distintos formatos que se verán más adelante. El más sencillo es usar nombres de colores comunes en inglés como "blue", "green", "black", "orange", "tomato", entre otros.
  • text-align: Sirve para definir la forma en la que queremos que el texto esté alineado. Los valores que puede tomar esta propiedad son: left, right, center y justify.
  • text-decoration: Sirve para que el texto aparezca tachado, subrayado o con una línea encima. Los valores que puede tomar esta propiedad son: none, underline, overline y line-through.

Subtítulo

Ejemplo de uso de propiedades de texto.

<!DOCTYPE html>
<html>
<head>

<title></title>
<meta charset="UTF-8">
<style>
h1
{
color: blue;
text-align: left;
text-decoration: underline;
}
h2
{
color: green;
text-align: center;
text-decoration: overline;
}
h3
{
color: tomato;
text-align: right;
text-decoration: line-through;
}
</style>

</head>
<body>

<h1>Título h1</h1>
<h2>Título h2</h2>
<h3>Título h3</h3>

</body>
</html>
