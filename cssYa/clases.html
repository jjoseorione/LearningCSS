Definición de estilos por medio de clases

Descripción

En CSS existe una herramiente muy útil llamada "clase". Estas clases no deben confundirse con las de la programación orientada a objetos, aunque son muy parecidas.

Hasta este momento sólo se ha estudiado la definición de estilos a elementos específicos de html (como *, body, p, h1, h2, div, etc.). Las clases sirven para crear estilos que pueden ser aplicadas después a cualquier elemento. De esta manera, no es necesario globalizar, por ejemplo, todos los elementos p de una sola manera y en su lugar podemos aplicar el estilo definido en una clase sólo a los elementos p que queramos modificar; de igual manera, el estilo definido en esa misma clase puede ser utilizado en otros elementos de HTML diferentes a p.

Para definir una clase solamente se necesita escribir en nuestra hoja de estilos el nombre que queremos que lleve nuestra clase antecedido por un punto, sin espacios. En seguida abrimos llaves y podemos definir nuestros estilos. Para invocar una clase en un elemento HTML sebe especificarse el atributo class="nombreClase" Las clases pueden usarse para definir estilos simples o complejos.

Ejemplo

En este ejemplo se crerá una clase llamada "resaltado" que servirá para resaltar texto. El texto será resaltado poniendo fondo verde y fuente gruesa (negrita).

<!DOCTYPE html>
<html>
<head>

<title></title>
<meta charset="UTF-8">
<style>
.resaltado
{
background-color: #39ff14;
font-weight: bold;
}
</style>

</head>
<body>

<h1>Título del texto con <span class="resaltado">palabras resaltadas</span></h1>
<p>El siguiente texto tiene algunas palabras <span class="resaltado">resaltadas</span> para mostrar la forma en que funciona la clase llamada <span class="resaltado">resaltado</span></p>
</body>
</html>
